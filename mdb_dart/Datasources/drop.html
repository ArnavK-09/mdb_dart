Deletes a specific Datasource identified by name.

This function removes the Datasource from the server. If the Datasource does not exist, it may result in an ObjectNotFound.

Parameters:

  • name: String The name of the datasource to be deleted.

Returns:

  • Future<void> A future that completes when the deletion is successful.

Example:

try {
  await datasources.drop('example_ds');
  print('Datasource dropped successfully.');
} catch (e) {
  print('Error: $e');
}

This example attempts to drop the Datasource named 'example_ds'

Implementation

Future<void> drop(String name) async {
  await client.api.delete('/datasources/$name');
}
