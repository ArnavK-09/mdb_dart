Mind class

Represents a Mind in the MindsDB project.

This class encapsulates the properties and methods associated with a Mind, allowing for interaction with the MindsDB API.

Properties:

  • name: String The name of the Mind.
  • modelName: String (optional) The name of the model associated with the Mind.
  • provider: String (optional) The provider for the Mind, such as OpenAI.
  • parameters: Map<String, dynamic> A map of parameters for configuring the Mind.
  • promptTemplate: String (optional) An optional template for prompts used by the Mind.
  • createdAt: String (optional) The timestamp when the Mind was created.
  • updatedAt: String (optional) The timestamp when the Mind was last updated.
  • datasources: List of either String/Datasource (optional) A list of data sources associated with the Mind.
