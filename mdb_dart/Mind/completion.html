Future<OpenAIChatCompletionModel> completion(
  1. String message
)

Sends a chat message to the Mind API and retrieves a chat completion response.

This asynchronous function takes a user's message as input and sends it to the API for processing. The response from the API includes generated text based on the input message.

Parameters:

  • message: String The message to be sent to the mind. This should be the user's input that the mind will generate a response for.

Returns:

  • Future<OpenAIChatCompletionModel> A Future that resolves to an OpenAIChatCompletionModel object, which contains the details of the generated chat completion including the generated text and additional metadata.

Example:

String userMessage = "List all data";
final response = await mind.completion(userMessage);
print('Chat completion ID: ${response.id}');
print('Generated response: ${response.choices.first.message.content?.first.text}');

In this example, a message is sent to the chat model, and the generated response is printed along with the chat completion ID.
