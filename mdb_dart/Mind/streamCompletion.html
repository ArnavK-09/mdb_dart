Streams chat completions from the Mind API based on a user's message.

This asynchronous function sends a user's message to the API and returns a stream of chat completion responses. This is useful for real-time applications where the response can be received in chunks as it is generated by the model.

Parameters:

  • message: String The message to be sent to the mind. This should be the user's input for which the mind will generate a response.

Returns:

  • Future<Stream<OpenAIStreamChatCompletionModel>> A Future that resolves to a stream of OpenAIStreamChatCompletionModel objects. Each object represents a chunk of the generated response from the chat model.

Example:

String userMessage = "Name your datasource";
await for (var response in (await mind1.streamCompletion(userMessage))) {
  print('Chunk received: ${response.choices.first.delta.content?.first?.text}');
}

In this example, a message is sent to the chat model, and the application listens for streamed responses, printing each chunk of the generated text as it arrives.

Implementation

Future<Stream<dart_openai.OpenAIStreamChatCompletionModel>> streamCompletion(
    String message) async {
  Map<String, dynamic> data = {
    'model': name,
    'messages': [
      {
        'role': 'user',
        'content': message,
      },
    ],
    'stream': true,
    ...parameters
  };

  final client = http.Client();
  final request = http.Request(
      'POST',
      Uri.parse(
          '${this.client.api.baseUrl.replaceAll("/api", "")}/chat/completions'))
    ..headers.addAll(this.client.api.headers())
    ..body = jsonEncode(data);
  final responseStream = client.send(request);

  return responseStream.asStream().asyncExpand((response) {
    return response.stream
        .transform(utf8.decoder)
        .transform(const LineSplitter())
        .where((line) =>
            line.startsWith('data:') && line.trim() != 'data: [DONE]')
        .map((line) => dart_openai.OpenAIStreamChatCompletionModel.fromMap(
            jsonDecode(line.substring(5).trim())))
        .handleError((e) {
      client.close();
      throw e;
    });
  });
}
